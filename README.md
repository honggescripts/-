# 汉字练字字帖生成器 Web 独立版
单文件 Python 实现的**田字格字帖 PDF 生成工具**，基于 Flask + Pillow 构建，无需拆分脚本，开箱即用，跨平台支持 Windows / macOS / Linux。

## 一、环境依赖
运行前请安装依赖库：
```bash
pip install flask pillow
```

## 二、核心功能
1. **三种生成模式**
   - 基础字帖：按单个汉字批量生成描红练习页；
   - 文章字帖：整段文本自动分行分页，适合古诗、短文抄写；
   - 多故事字帖：以空行分隔多篇内容，独立分页排版。

2. **田字格样式**
   标准田字格布局，自带中线、斜线虚线辅助线，区分示范字与浅灰色描红字，符合练字规范。

3. **高度自定义配置**
   - 字体：支持上传自定义 `.ttf/.otf` 字体，无上传则自动调用系统中文字体；
   - 页面：支持 A4 纸张、自定义毫米尺寸、边距、打印 DPI；
   - 布局：可设置每行格子数量、字体占比、格子间距、行间距；
   - 样式：自由调整边框颜色、辅助线颜色、描红灰度深浅；
   - 标题：自定义字帖顶部标题，也可隐藏。

4. **使用体验**
   - 启动程序自动打开本地网页端，可视化操作；
   - 输入内容+配置参数，一键生成并下载 PDF；
   - 程序退出自动清理临时文件，无残留。

## 三、使用方法
1. 安装依赖后，直接运行主 Python 脚本；
2. 浏览器会自动访问 `http://127.0.0.1:5000` 进入操作页面；
3. 填写练字文本、按需修改各项参数；
4. 选择对应生成模式，等待生成后自动下载 PDF 文件。

## 四、特点
- 单文件独立运行，无额外资源依赖；
- 输出标准 PDF，适配家用/办公打印机；
- 参数丰富，可适配幼儿启蒙、学生练字、教学素材等多种场景。
